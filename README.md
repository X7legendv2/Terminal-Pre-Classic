# Terminal Pre-Classic

The very initial version of the terminal, named after the old versions of Minecraft.  
Under development. Soon there will be the first console commands.

---

**RUS / РУС**

Самая начальная версия терминала, названная в честь старых версий Minecraft — от Pre‑Classic до Beta.  
Находится в разработке. Скоро здесь появятся первые консольные команды.

---

## 📜 История версий

### v0.0.1 — Тестовая (Pre‑Classic Alpha)
Самая первая версия.  
- Логин и регистрация  
- Проверка данных  
- Базовая структура  

---

### v0.1.0 — Pre‑Classic (первая рабочая)
Появился игровой цикл.  
- Игровой цикл  
- Базовые команды: `whoami`, `plasma`, `bit`, `help`, `exit`  
- Регистрация  

---

### v0.2.0 — Pre‑Classic (обновление)
Добавлены новые команды.  
- Команда `time` — показывает время и месяц  
- Исправлены ошибки  
- Улучшен код  

---

### v0.3.0 — Pre‑Classic Release (стабильный релиз)
Финальная стабильная версия.  
- 7 команд: `whoami`, `plasma`, `bit`, `help`, `time`, `clear`, `exit`  
- Очистка экрана (`clear`)  
- Исправлены все известные ошибки  
- Полностью стабильная версия  
- Грамматические ошибки исправлены
